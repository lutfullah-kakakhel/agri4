üÜò Troubleshooting

Windows Defender Warning:
If Windows Defender shows a security warning, click "More info" and then "Run anyway". This is normal for new applications that haven't been downloaded many times yet.
Missing DLL Files:
If you get "missing DLL" errors with the portable version, make sure you extracted ALL files from the ZIP archive, not just the .exe file.
Internet Connection:
The app requires an internet connection for maps and weather data. Make sure you're connected to the internet when using the app.
Administrator Rights:
The MSI installer may require administrator rights. Right-click the MSI file and select "Run as administrator" if needed.
